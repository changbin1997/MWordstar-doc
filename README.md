在确保安装 Node.js 的情况下通过命令行进入项目目录可使用 npm 操作文档。

安装所需依赖：

```bash
npm install
```

在本地运行项目：

```bash
npm run start
```

运行成功后在浏览器中输入 `localhost:3000` 访问。

编译为静态页面：

```bash
npm run build
```

编译后生成的页面就在 `build` 目录中。
