开始使用

响应式设计

主题可以兼容 PC。Pad。手机。IE 浏览器最低可兼容到 IE9。部分动画可能需要 IE10 以上才能完美兼容。

代码高亮

主题支持 30 多种语言的代码高亮，代码高亮的主题是 VS2015 的暗色主题，和 Visual Studio Code 的默认主题差不多。

Accessibility

主题针对键盘和屏幕阅读器做了大量优化，能准确传达需要朗读的内容和信息，对于盲人来说，使用标准的屏幕阅读器基本可以无障碍操作。

丰富的设置选项

主题提供了丰富的设置选项，您可以根据喜好调整主题的配色、元素风格、侧边栏组件、以及个区域的显示和隐藏等。

Emoji表情

在评论区包含一个 Emoji 表情面板，目前共有 1466 个表情，这些表情都是按照分类动态加载的，您不用担心性能问题。

文章归档

主题提供了多种文章归档方案，其中包括按月份的侧边栏归档组件，按日期的侧边栏日历组件，按时间线的独立页归档模板。
